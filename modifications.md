# List of Fedora modification to GNOME

As of Fedora 42:

- Different desktop-backgrounds
- gnome-shell extensions
    - logo at the bottom of the gdm login screen
- Default wallpaper
- Epiphany -> Firefox 
- Console -> Ptyxis
    - a11y enabled in Ptyxis, disabled upstream
    - nautilus patch to launch ptyxis
- no GNOME Music
- enable-smartcard-authentication=false
    - This needs to be upstreamed in GDM among with other fixes
- mutter experimental features scale-monitor-framebuffer and xwayland-native-scaling enabled
- GNOME Software configuration
- Branded gnome-tour logo
- Different default mimetypes. GNOME Session 49 will include a list upstream
